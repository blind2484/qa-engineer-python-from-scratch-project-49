### Description

Five simple math games:

brain-even game: determine whether a number is even or not

brain-calc game: calculate a simple mathematical expression

brain-gcd game: find the greatest common divisor

brain-progression game: insert a missing element of the progression

brain-prime game: determine whether a number is prime
